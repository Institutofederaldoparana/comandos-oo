# Estruturas de Decisão

## if

![PHP](https://img.shields.io/badge/Decisão-if-2196F3?style=for-the-badge)

```php
if
```

Executa um bloco de código caso uma condição seja verdadeira.

---

## else

![PHP](https://img.shields.io/badge/Decisão-else-03A9F4?style=for-the-badge)

```php
else
```

Executa um bloco alternativo quando a condição do `if` é falsa.

---

## switch

![PHP](https://img.shields.io/badge/Decisão-switch-00BCD4?style=for-the-badge)

```php
switch
```

Seleciona um bloco de código entre vários casos.

---

## case

![PHP](https://img.shields.io/badge/Decisão-case-009688?style=for-the-badge)

```php
case
```

Representa uma opção dentro do `switch`.

---

## break

![PHP](https://img.shields.io/badge/Controle-break-FF9800?style=for-the-badge)

```php
break
```

Interrompe um `switch` ou um laço de repetição.

---

# Estruturas de Repetição

## while

![PHP](https://img.shields.io/badge/Loop-while-9C27B0?style=for-the-badge)

```php
while
```

Executa um bloco enquanto a condição for verdadeira.

---

## do...while

![PHP](https://img.shields.io/badge/Loop-do...while-673AB7?style=for-the-badge)

```php
do {

} while (condição);
```

Executa o bloco pelo menos uma vez.

---

## for

![PHP](https://img.shields.io/badge/Loop-for-E91E63?style=for-the-badge)

```php
for
```

Executa um bloco um número determinado de vezes.

---

# Programação Orientada a Objetos

## class

![PHP](https://img.shields.io/badge/POO-class-607D8B?style=for-the-badge)

```php
class
```

Define uma classe.

---

## public

![PHP](https://img.shields.io/badge/POO-public-4CAF50?style=for-the-badge)

```php
public
```

Define um atributo ou método acessível de qualquer lugar.

---

## private

![PHP](https://img.shields.io/badge/POO-private-F44336?style=for-the-badge)

```php
private
```

Define um atributo ou método acessível apenas dentro da própria classe.

---

## public function

![PHP](https://img.shields.io/badge/POO-public_function-8BC34A?style=for-the-badge)

```php

{

}
```

Declara um método público da classe.
